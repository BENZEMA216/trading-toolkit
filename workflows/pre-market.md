# Pre-Market Workflow

盘前分析流程，每个交易日开盘前执行。

## Flow

```
1. 五大指标检查 (/stock-analyst)
   ├── VIX < 20 → 正常
   ├── VIX 20-27 → 谨慎
   └── VIX > 27 → 防御模式

2. 持仓个股检查
   ├── 价格 vs 支撑/阻力位
   ├── 隐含波动率变化
   └── 盘前异动

3. 交易计划 (if needed)
   ├── /b216 评估拟交易
   ├── 21 位大师投票
   └── B216 Score > +0.3 → 执行
```

## Checklist

- [ ] 五大指标截图/记录
- [ ] VIX 位置判断市场温度
- [ ] 持仓个股有无异动
- [ ] 今日是否有交易计划
- [ ] 如有，B216 Score 是否支持
